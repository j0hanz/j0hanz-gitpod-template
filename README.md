## âš¡ Linters & Formatters

### **djlint**: Django Template Linter

- **Config**: In `pyproject.toml`
- **Usage**:

  ```bash
  djlint templates/
  djlint --reformat templates/
  ```

### **ruff**: Fast Python Linter

- **Config**: In `pyproject.toml`
- **Usage**:

  ```bash
  ruff check .
  ruff check . --fix
  ```
